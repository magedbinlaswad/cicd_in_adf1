# Azure Data Factory CI/CD Automation - Project 1

## Description
This repository demonstrates implementing **CI/CD pipelines** for **Azure Data Factory (ADF)** using **GitHub Actions**.  
It focuses on automating deployment, version control, and pipeline orchestration for multiple environments.

## Key Features
- Automated deployment of ADF pipelines from DEV to PROD
- Version-controlled infrastructure as code (ARM templates)
- Managed datasets, pipelines, linked services, and integration runtimes
- Workflow automation using GitHub Actions

## Tech Stack
- Azure Data Factory
- GitHub Actions (CI/CD)
- ARM Templates
- JSON configurations

## Usage
1. Fork or clone the repository
2. Connect GitHub Actions to your Azure subscription
3. Trigger workflows to deploy ADF pipelines

## Notes
This is a production-grade setup for learning automated CI/CD with Azure Data Factory.
